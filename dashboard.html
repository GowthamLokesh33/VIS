UPI123

UPI123 is mainly implemented for the Rural Area People's and in Internet down situations. For this UPI123 the From User and the End User don't need the Internet Connectivity, only the middleware applications (IVR and Infrasoft API and NPCI) need the Internet Connnectivity. The maximum tranasction history is Rupees 2000 as of now, in future if needed to be increased or decresed using the property file configurations we can modify it. To use the UPI123 service, 1st need to Register to the UPI123. After the Registration process, we can able to do the Fund Transfer for both On-Us & Off-Us. On-Us (Fund Transfer between the Same Banks) and Off-Us (Fund Transfer to Other Banks) check the account Current Balance and check the Last 5 Transactions History. MPIN & UPIN can also be changed and UPI123 service can also be De-Registered.
